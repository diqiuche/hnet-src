# 安装

```
编译libhnet.so libhnet.a
```

* 环境最低要求

    * Linux2.6.x（Linux2.4.x + epoll补丁）

    * g++ (GCC) 4.8.x

* 编译

    * 解压hnet.tar.gz到任一目录中，以 /usr/local/hnet 为例。
        * cd /usr/local
        * tar -zxvf hnet.tar.gz
        * cd hnet/core
        * make
        * make install
